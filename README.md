# Trader Performance vs Market Sentiment

## Objective

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Key Findings

- Extreme Greed periods showed the highest average profitability.
- Fear periods exhibited the highest trading activity and largest average position sizes.
- Trader behaviour changed with market sentiment.
- Infrequent traders achieved higher average profitability than frequent traders.
